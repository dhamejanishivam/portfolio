Heart Disease Predictor

View Source Code   
This Heart Disease Prediction System utilizes a logistic regression model trained on historical medical data to predict the risk of heart disease based on user inputs such as age and cholesterol levels. It features a machine learning model accessible via a Python script (main.py) for direct predictions, or a web interface (requiring MySQL) for user-friendly interactions. While the repository doesn't include a MySQL database or sample data, the model can be used independently without the web setup. The project structure includes the model folder, HTML templates, and assets for styling. It requires Python 3.x and libraries like pandas, scikit-learn, and Flask.

Pendrive Virus

View Source Code   
This project demonstrates a Python script capable of copying selected files (PDF, text, documents, images) from a computer to a USB drive and retrieving stored WiFi passwords from the system. It also gathers system details such as IP addresses and victim PC information. An additional feature allows for the deletion of all files after execution, offering a cleanup option. The script works by identifying files based on extensions, using system commands to retrieve WiFi passwords, and storing the collected information in a details.txt file. It requires a Windows OS, admin privileges, and Python installed on the pendrive.
